# LoFTP Wiki

LoFTP is a dual-pane file manager for macOS focused on practical work with local files, FTP, FTPS, and SFTP servers.

This wiki is the quick reference for:

- what LoFTP currently does
- how to install and use it
- how licensing and project support work
- how AI, Codex, and VS Code fit into the workflow
