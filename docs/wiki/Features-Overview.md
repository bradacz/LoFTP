# Features Overview

## File Management

- dual-pane file browsing
- local and remote file views
- local-to-local copy between panes
- rename, delete, create folder, and common file operations

## Server Access

- saved FTP profiles
- saved FTPS profiles
- saved SFTP profiles
- one-click reuse of saved server connections

## Transfer Tools

- upload files and directories
- download files and directories
- transfer queue with progress and cancellation
- overwrite, resume, timestamp, permission, symlink, verification, and directory creation options

## Compare and Search

- directory comparison between the left and right pane
- local filesystem search by file name
- local filesystem search by file content

## Viewer and Editor

- quick view for text files
- quick view for images
- quick view for PDF files
- quick view for binary files in hex mode
- built-in editor for local text files

## Archives

- browse ZIP archives
- browse TAR, TAR.GZ, and TGZ archives
- extract archives
- create ZIP archives from selected local files

## Desktop Integration

- quick access to mounted volumes
- quick access to iCloud and common CloudStorage folders
- light and dark themes
- localization in multiple languages
- in-app updater flow
