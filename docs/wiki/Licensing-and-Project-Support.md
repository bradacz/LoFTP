## What The Payment Means

LoFTP works without functional restrictions.

The payment shown in the application is not the purchase of a locked full version. It is a voluntary contribution to continued development and related support or activation services.

Current displayed contribution amount:

- 25 EUR

## Branding

The source code license and project branding are handled separately.

Repository branding note:

- [TRADEMARKS.md](https://github.com/bradacz/LoFTP/blob/main/TRADEMARKS.md)

In practice this means:

- the code is MIT licensed
- the LoFTP name, logo, and branding are not automatically granted for misleading reuse

## Why This Matters

This structure keeps the project open source while still allowing:

- voluntary support payments
- long-term maintenance funding
- protection of the official project identity
